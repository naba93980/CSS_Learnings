specificity rule --- 

 id selector has highest value.
 universal selector has lowest value. 
 

--10,000 point to !important
--1000 to inline
--100 to id
--10 to class, attribute or pseudo-class
--1 for element selector and pseudo-elements
--0 to universal selector


If one element is styled by more than one rule, which style will be applicable is decided by specifity-- 

!important > inline > external according to specificity
