every element in css is a rectangular box, and how that box gonna behave depends on the display property */
every element has some default value for all the property

block-
1.starts from new-line
2.takes full width
3.height and width can be applied
 
inline-
1.doesnot start from new line
2.does not take full width
3.height and width cant be applied (basically inline is rigid)


inline-block-
1.doesnot start from new line 
2.does not take full width 
3.height and width CAN be applied


none


inline or block is the mostly used default values for display properties

we can change display property of elements inline to block or vice versa
